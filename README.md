BMO - Proyecto [BMO-VAQUERO]
Este repositorio contiene una versión modificada del modelo "Raspberry Pi 4 case BMO Adventure Time", diseñado originalmente por kyleschieffer y disponible en Printables.

Licencia y Atribución
Este proyecto se distribuye bajo la licencia Creative Commons Attribution 4.0 International (CC BY 4.0). De acuerdo con los términos de esta licencia, se especifica que el presente trabajo es una obra derivada que contiene modificaciones respecto al diseño original de kyleschieffer.

Descripción de cambios
Se realizaron ajustes en los archivos STL originales para adaptar el chasis a una Raspberry Pi 5 y una pantalla de 5'', agregarle un espacio para una bocina y modificaciones en el tamaño de las piezas
